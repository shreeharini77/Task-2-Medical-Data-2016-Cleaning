# Task 2: Data Cleaning & Missing Value Handling

## Dataset
Medical Appointment No Shows (Kaggle)

## Objective
To clean the dataset, check for missing values, and improve data quality
before analysis or machine learning.

## Steps Performed
- Loaded the dataset using pandas
- Checked missing values using isnull().sum()
- Visualized missing values using a bar plot
- Removed rows with invalid age values (negative age)
- Standardized column names
- Converted date columns to datetime format
- Saved the cleaned dataset

## Outcome
A clean and validated dataset ready for further analysis.
